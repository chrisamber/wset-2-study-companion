# Contributing

This is a personal case-study project built for one person's WSET Level 2 exam, not a general-purpose tool actively seeking contributors. Scope decisions (what grapes/content to include, what to leave out) are deliberate — see the [README's "Why it exists"](README.md#why-it-exists) and `docs/superpowers/` for the reasoning behind recent changes.

That said:

- **Bug reports and typos:** open an issue or a small PR — happy to take a look.
- **Feature requests / content expansion:** open an issue to discuss first. Larger changes (e.g. new grape varieties, new quiz content) need real WSET-spec sourcing, not invented facts, so these take more than a quick PR.
- **Before submitting a PR:** run `npm run lint`, `npm test`, and `npm run build` locally — CI runs the same checks.
